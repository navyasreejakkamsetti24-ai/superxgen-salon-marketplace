# SuperXgen Salon Marketplace

## Overview
SuperXgen Salon Marketplace is an AI-powered platform that connects customers with salons and beauty service providers.

## Problem
Customers struggle to find trusted salons and book appointments easily.
Salon owners need better online visibility and booking management.

## Solution
A digital marketplace where users can:
- Discover salons
- Compare services
- Book appointments
- Make online payments
- Get AI-powered recommendations

## Features
- City-wise salon listings
- Online booking
- User dashboard
- Salon owner dashboard
- Search and filters
- AI chatbot support

## Tech Stack
- Next.js
- Firebase
- Tailwind CSS
- Vercel
